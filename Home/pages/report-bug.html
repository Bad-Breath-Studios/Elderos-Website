Report a Bug

Help us squash bugs and improve the game.

Found something broken, glitched, or not working as expected? Submit a report below and our team will investigate. The more detail you provide, the faster we can fix it.

Alternative Reporting

You can also report bugs through:

  • Discord — Use the bug-reports channel in our Discord server.
  • In-game — Use the ::bug command to submit a quick report from within the game.

Bug Reporting Guidelines

  • Be specific — "combat is broken" is less helpful than "Dharok's greataxe special attack hits 0 every time."
  • Include steps to reproduce — If we can't trigger the bug, it's much harder to fix.
  • One bug per report — Multiple issues in one report can get confusing.
  • Don't exploit bugs — If you find an exploitable bug, report it privately via Discord DM to a staff member. Exploiting bugs can result in a ban.
